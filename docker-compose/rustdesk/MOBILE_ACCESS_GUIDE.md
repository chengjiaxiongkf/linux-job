# 移动端访问RustDesk详细指南

## 通过手机浏览器访问局域网IP+端口连接RustDesk

### 方法一：使用Web界面（推荐）

#### 1. 局域网Web访问
- 在手机浏览器地址栏输入：`http://你的电脑局域网IP:80`
- 例如：`http://192.168.1.100`
- 这将访问RustDesk的Web界面

#### 2. 公网访问（如果有公网IP）
- 在手机浏览器输入：`http://你的公网IP:80`
- 或使用域名：`http://yourdomain.com`

#### 3. Web界面功能
- 远程桌面控制
- 文件传输
- 剪贴板同步
- 屏幕截图
- 多显示器支持

### 方法二：使用RustDesk移动端APP

#### 1. 下载APP
- **Android**: 在Google Play商店搜索"RustDesk"或下载APK
- **iOS**: 在App Store搜索"RustDesk"

#### 2. 配置自建服务器
打开APP → 设置 → 网络 → 自建服务器：
- **ID服务器**: 你的局域网IP（如：192.168.1.100）
- **中继服务器**: 同上
- **密钥**: 使用部署时生成的密钥
- **API服务器**: 留空或填写你的服务器地址

#### 3. 连接步骤
1. 在电脑上打开RustDesk客户端
2. 在手机APP中输入电脑的RustDesk ID
3. 点击连接
4. 在电脑上确认连接请求

## 局域网IP获取方法

### Windows系统
```cmd
ipconfig
```
查找"IPv4地址"，通常是192.168.x.x格式

### Linux/Mac系统
```bash
ip addr show
# 或
ifconfig
```

### 手机端查看局域网IP
- **Android**: 设置 → 关于手机 → 状态 → IP地址
- **iOS**: 设置 → WiFi → 点击当前网络 → IP地址

## 端口配置说明

### 默认端口映射
| 服务 | 容器端口 | 主机端口 | 协议 | 用途 |
|------|----------|----------|------|------|
| Web界面 | 21114 | 80 | TCP | Web访问 |
| API服务 | 21115 | 21115 | TCP | API接口 |
| ID注册 | 21116 | 21116 | TCP/UDP | 客户端注册 |
| 中继服务 | 21118 | 21118 | TCP | 数据传输 |

### 局域网访问端口
- **Web界面**: `http://局域网IP:80`
- **直接API**: `http://局域网IP:21114`

## 移动端优化设置

### 触摸控制优化
1. **触摸模式**: 在Web界面启用触摸模式
2. **手势支持**: 
   - 单指点击 = 鼠标左键
   - 双指点击 = 鼠标右键
   - 双指滑动 = 滚轮
   - 三指滑动 = 拖拽

### 显示优化
1. **自适应屏幕**: 自动调整远程桌面分辨率
2. **缩放控制**: 双指捏合缩放
3. **全屏模式**: 横屏获得更好体验
4. **画质调节**: 根据网络调整画质

### 网络优化
1. **局域网优先**: 同一网络下优先使用局域网IP
2. **带宽检测**: 自动检测网络速度
3. **压缩算法**: 启用数据压缩减少流量

## 常见问题解决

### 无法连接局域网IP
1. **检查防火墙**: 确保电脑防火墙允许相关端口
2. **确认IP地址**: 使用`ping`命令测试连通性
3. **端口占用**: 检查端口是否被其他程序占用

### 移动端Web界面显示异常
1. **浏览器兼容性**: 使用Chrome/Safari最新版
2. **清除缓存**: 清除浏览器缓存重试
3. **屏幕旋转**: 尝试横屏模式

### 连接速度慢
1. **网络质量**: 确保WiFi信号强度
2. **带宽限制**: 检查是否有带宽限制
3. **画质调整**: 降低远程桌面画质

## 安全建议

### 局域网安全
1. **强密码**: 为RustDesk设置强密码
2. **访问控制**: 限制可访问的IP地址
3. **定期更新**: 保持RustDesk版本更新

### 移动端安全
1. **APP权限**: 仅授予必要权限
2. **安全连接**: 优先使用加密连接
3. **及时断开**: 使用完毕后断开连接

## 高级配置

### 自定义端口
修改`docker-compose.yml`中的端口映射：
```yaml
ports:
  - "自定义端口:21114"  # Web界面
  - "自定义端口:21115"  # API服务
```

### SSL/HTTPS配置
1. 获取SSL证书
2. 修改nginx.conf配置
3. 使用`https://局域网IP`访问

### 多设备管理
1. 为每个设备设置不同的显示名称
2. 使用设备分组功能
3. 设置访问权限控制

## 性能优化

### 局域网优化
- 使用5GHz WiFi频段
- 确保路由器性能足够
- 减少网络干扰

### 移动端优化
- 关闭不必要的后台应用
- 保持设备散热
- 定期清理缓存

通过以上配置，你可以轻松通过手机浏览器访问局域网中的RustDesk服务，实现便捷的远程控制功能。
